## 課題 1

- NULL と比較する演算子は、`<=>`以外は全て NULL となる
  NULL = 0 は NULL
  NULL = NULL は NULL
  NULL <> NULL は NULL

- 論理演算は TRUE(1)、FALSE(0)、NULL のいずれか
  NULL AND TRUE は NULL
  NULL AND FALSE は FALSE(0)
  NULL OR TRUE は TRUE(1)
